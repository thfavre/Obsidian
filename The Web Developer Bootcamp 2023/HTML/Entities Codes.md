#HTML 

```html
&#9824;
```
Used to write special characters.
- Starts with `&` and Ends with `;`
Lists : https://html.spec.whatwg.org/multipage/named-characters.html

## Example :
How to write : "1 < 6"
```html
<h1> 1 < 6 </h1>
```
Error, instead :
```html
<h1> 1 &lt; 6 </h1>
```
